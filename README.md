# Gerador de Números da Lotaria

Esta é uma aplicação Android simples desenvolvida em Kotlin que permite aos utilizadores gerar um conjunto de números aleatórios para a lotaria.

## Funcionalidades

*   **Geração de Números:** Gera 6 números aleatórios únicos entre 0 e 42.
*   **Personalização:** O utilizador pode inserir o seu nome, que é exibido no ecrã de resultados.
*   **Partilha:** Permite partilhar facilmente os números gerados através de outras aplicações, como email, SMS ou redes sociais.

## Como Utilizar

1.  **Ecrã Principal:**
    *   Insira o seu nome no campo de texto.
    *   Clique no botão "Gerar Números Aleatórios da Lotaria".

2.  **Ecrã de Resultados:**
    *   Visualize os 6 números da sorte que foram gerados para si.
    *   Clique no botão "Partilhar Resultado" para enviar os números a um amigo.

## Tecnologias Utilizadas

*   **Linguagem:** Kotlin
*   **Arquitetura:** Aplicação de duas atividades simples com passagem de dados através de `Intents`.
*   **Bibliotecas Android:**
    *   AndroidX (AppCompat, ConstraintLayout, Core-KTX)
    *   Material Design Components
